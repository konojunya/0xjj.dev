## How it works

レスラーアトラクタは、1976 年に Otto Rössler が提案した**カオス力学系**です。ローレンツアトラクタよりもシンプルな構造を持ち、x-y 平面上で平らな螺旋を描きながら、時折 z 方向に大きく跳ね上がる「折り返し」が特徴的です。「最も単純なカオス系の一つ」として、カオスの基本的な振る舞い — ストレッチとフォールド — を明瞭に観察できます。

### 微分方程式

レスラー系は以下の 3 つの連立常微分方程式で定義されます。

```
dx/dt = -y - z
dy/dt = x + ay
dz/dt = b + z(x - c)
```

古典的なパラメータは a = 0.2、b = 0.2、c = 5.7 です。x と y の項が螺旋運動を、z の項（x - c で符号が切り替わる非線形項）が特徴的な折り返しを生み出します。

### フラグメントシェーダーでの描画

ローレンツアトラクタと同様に、WebGL2 フラグメントシェーダー内でオイラー法による数値積分をピクセルごとに実行しています。

```glsl
vec3 d = vec3(
  -pos.y - pos.z,          // dx/dt
  pos.x + A * pos.y,       // dy/dt
  B + pos.z * (pos.x - C)  // dz/dt
);
pos += d * dt;
```

ウォームアップフェーズ（200〜600 ステップ）で初期値 `(1, 1, 1)` からアトラクタ上に到達させた後、300〜600 ステップの軌跡を描画します。ウォームアップの長さを時間で変化させることで、軌跡がアトラクタ上を「流れる」アニメーションを実現しています。

### ローレンツアトラクタとの違い

| | ローレンツ | レスラー |
|---|---|---|
| 形状 | 蝶の翅のような 2 つのローブ | 平らな螺旋 + 折り返し |
| カオスの種類 | 2 つのローブ間の不規則な遷移 | ストレッチ & フォールド |
| 非線形項 | `xz`, `xy` | `xz` のみ |
| 視覚的な特徴 | 左右対称 | 非対称、z 方向のスパイク |

### カラーとグロウ

軌跡は HSV カラースペースで虹色に塗られ、`fract(phase * 2.0 + u_time * 0.05)` で色が時間とともに流れます。線自体は `smoothstep` でシャープに、周囲を `exp(-d²)` のガウシアンでぼんやり光らせるグロウエフェクトを加えています。

```glsl
vec3 col = hsv2rgb(fract(phase * 2.0 + u_time * 0.05), 0.8, 1.0);
colorAcc += col * (line + glow * 0.12);
```

最終出力は Reinhard トーンマッピングとガンマ補正で仕上げています。
