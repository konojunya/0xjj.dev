## How it works

Audio Reactor は、マイクから取得したリアルタイム音声を FFT で周波数分解し、その Bass・Mid・Treble・全体エネルギーで **レイマーチング**された万華鏡空間を駆動する VJ ビジュアライゼーションです。

### レイマーチングとは

レイマーチングは、カメラからスクリーンの各ピクセルに向かってレイ（光線）を飛ばし、**距離関数**（SDF: Signed Distance Field）を使って最寄りの物体表面までの距離を測りながら進む描画手法です。ポリゴンメッシュを使わずに、数式だけで複雑な 3D 形状を描画できます。

このデモでは WebGL2 フラグメントシェーダー上で **80 ステップ**（モバイルは 40 ステップ）のレイマーチを行い、球体・トーラス・八面体をスムーズモーフィングで合成したシーン SDF を評価しています。

### 音声解析パイプライン

Web Audio API の `AnalyserNode` で FFT を実行し、周波数ビンを 3 帯域に分割します。

```ts
// FFT ビンの帯域分割
const bassEnd = Math.floor(binCount * 0.12);
const midEnd  = Math.floor(binCount * 0.40);
// 0〜12% → Bass, 12〜40% → Mid, 40〜100% → Treble
```

ビート検出は「瞬間エネルギーが EMA 平滑値の 1.35 倍を超えたらビート」というシンプルなしきい値方式です。

### 万華鏡フォールド

レイの方向ベクトルを極座標に変換し、角度を `N` 等分に折りたたむことで万華鏡効果を生みます。分割数は音に応じて動的に変化します。

```glsl
float folds = complexity + bass * 4.0 + mid * 3.0 + treble * 2.0 + energy * 3.0;
float angle = atan(rd.y, rd.x);
float sector = 6.2831853 / folds;
angle = mod(angle, sector) - sector * 0.5;
rd.xy = vec2(cos(angle), sin(angle)) * length(rd.xy);
```

音が大きいほど分割数が増え、万華鏡がより複雑に砕けていきます。

### エネルギーゲーティング

無音時に真っ暗にならないよう、`smoothstep(0.0, 0.04, energy)` で 0→1 のゲートを作り、アニメーション速度・彩度・輝度すべてをこのゲートで制御しています。無音時はほぼモノクロで微動、音が入ると一気にフルカラーで爆発する演出になります。

### ビートフラッシュ

ビートを検出すると白いフラッシュと色収差（R/B チャンネルを UV 方向にオフセット）を一瞬だけ適用します。

```glsl
if (u_beat > 0.01) {
  float flash = u_beat * u_beat;
  color += flash * 0.3;
  // 色収差: R と B を UV 方向にずらす
}
```

### ポストプロセス

最終出力は Reinhard トーンマッピング `color / (1.0 + color)` とガンマ補正 `pow(color, 0.9)` で仕上げています。ビネットは無音時に強く（画面端を暗く）、音が大きいと弱くなり、視野が広がる感覚を演出します。
