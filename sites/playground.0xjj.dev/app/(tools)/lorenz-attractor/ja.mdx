## How it works

ローレンツアトラクタは、1963 年に気象学者 Edward Lorenz が大気対流のモデル化で発見した**カオス力学系**です。3 つの単純な微分方程式から、「バタフライ効果」で有名な蝶の形をした軌道が生まれます。初期値のわずかな違いが指数的に増幅され、長期予測が不可能になる — カオス理論の象徴的な例です。

### 微分方程式

ローレンツ系は以下の 3 つの連立常微分方程式で定義されます。

```
dx/dt = σ(y - x)
dy/dt = x(ρ - z) - y
dz/dt = xy - βz
```

古典的なパラメータは σ = 10、ρ = 28、β = 8/3 です。このデモでは**フラグメントシェーダー**内で毎ピクセル、オイラー法で数百ステップの数値積分を実行しています。

```glsl
vec3 p = vec3(-8.0, 8.0, 27.0); // 初期値
float dt = 0.01;

for (int i = 0; i < steps; i++) {
  vec3 d = vec3(
    SIGMA * (p.y - p.x),
    p.x * (RHO - p.z) - p.y,
    p.x * p.y - BETA * p.z
  );
  p += d * dt;
}
```

### 流線の描画

各ピクセルについて、積分した軌跡の各線分（前のステップと現在のステップを結ぶ線）までの**最短距離**を計算し、近ければ明るく光らせます。線は `smoothstep` でシャープなエッジ、`exp(-d²)` のガウシアンでぼんやりしたグロウを加えています。

```glsl
float line = smoothstep(lineW, lineW * 0.1, dist);
float glow = exp(-dist * dist / (glowW * glowW));
```

### 虹色の流れ

軌跡上の位置 `phase` と `u_time` を使って HSV カラーを生成し、色が軌跡に沿って「流れる」アニメーションを作っています。

```glsl
vec3 col = hsv2rgb(fract(phase * 2.0 + u_time * 0.05), 0.8, 1.0);
```

`phase * 2.0` で軌跡を 2 周分の色相で塗り、`u_time * 0.05` でゆっくりと色が流れていきます。

### 3D 回転

3D 座標を 2D スクリーンに投影する際、方位角と仰角の回転行列を適用しています。ポインター位置と経過時間に応じて自動回転し、蝶の形を様々な角度から観察できます。

### パラメータ

- **ズーム** — アトラクタの表示スケール
- **回転** — 自動回転の速度
- **仰角** — カメラの上下方向の角度
- **軌跡** — 描画するステップ数（250〜500）。多いほど軌道が長くなる
- **太さ** — 線の幅とグロウの半径
- **速度** — 流れるアニメーションの速度
